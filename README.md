# Oscar Apple Portfolio

Next.js portfolio med Apple-inspirerad design, 3D-bakgrund, glassmorphism, tema, AI-chat i demoläge, Sanity-förberedelse, Stripe-förberedelse och kodlåsta projektkort.

## Starta lokalt

```bash
npm install
cp .env.example .env.local
npm run dev
```

Öppna: http://localhost:3000

## Kod för låsta projektkort

```txt
OSCAR2026
```

## Publicera på Vercel

1. Ladda upp projektet till GitHub.
2. Gå till Vercel.
3. Importera GitHub-repot.
4. Klicka Deploy.
